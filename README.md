## Project Description
The Placement Management System (PMS) is a web application that helps manage the placement process in a college. It connects students, companies, and administrators in one platform.

Students can apply for jobs, companies can post job openings, and the admin can track placement statistics.

## Tech Stack
Frontend: React
Backend: Node.js + Express
Database: MySQL

## Modules
- Authentication
- Student Portal
- Company Portal
- Job Application System
- Interview Management
- Admin Dashboard
